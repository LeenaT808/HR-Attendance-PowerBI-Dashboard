# HR Attendance Analytics Dashboard (Power BI)

## Project Overview

This project analyzes employee attendance data to monitor workforce patterns and identify absenteeism, late arrivals, and early exits.
The dashboard helps HR teams track attendance performance and quickly identify employees with frequent attendance issues.

## Tools Used

* Power BI
* Excel

## Key KPIs

* Attendance Rate
* Total Present
* Missing Punch
* Late Arrivals
* Early Exit

## Dashboard Features

* Monthly attendance trend
* Late arrival and early exit analysis
* Employee-wise late arrival count
* Employee-wise early exit count
* Absenteeism analysis by day
* Attendance status distribution (Full Day, Half Day, Missing Punch, etc.)

## Insights

* Overall attendance remained stable around **95–97%**.
* Late arrivals are concentrated among a small group of employees.
* Attendance records are mostly **Full Day entries**.
* Certain days show slightly higher absenteeism compared to others.
